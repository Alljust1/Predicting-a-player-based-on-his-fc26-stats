Project Overview / Введение

Цель проекта:
Прогнозирование основной позиции игрока на футбольном поле (FIFA 26) на основе его статистических характеристик.

Данные:

Источник:https://www.kaggle.com/datasets/rovnez/fc-26-fifa-26-player-data

Количество строк: 18405 игроков

Количество признаков: 110 (числовые и категориальные)

Проблема: предсказать main_position игрока (CAM, CB, CDM, CM, GK, LB, LM, LW, RB, RM, RW, ST)

2. Data Exploration / Исследование данных

Основные моменты:

Пропуски: 149035

Типы данных: float64 – 16, int64 – 48, object – 46

Основная переменная: main_position (извлечена из player_positions)

Распределение игроков по позициям: визуализировано с помощью bar plot

Пример графика распределения игроков по позициям:

df["main_position"].value_counts().plot(kind="bar", figsize=(12,4))

3. Data Preprocessing / Предобработка данных

Выбор числовых признаков (int64, float64) для модели

Кодирование целевой переменной main_position с помощью LabelEncoder

Обработка пропусков:

Использован SimpleImputer(strategy='mean') для числовых признаков

Разделение данных на train/test: 80% / 20%

4. Modeling / Моделирование

Модели:

Random Forest Classifier

Logistic Regression

XGBoost Classifier

Процесс:

Обучение моделей на train set

Предсказания на test set

Оценка точности (accuracy_score)

Построение отчета классификации (classification_report)

Визуализация матрицы ошибок (Confusion Matrix)

Визуализация важности признаков для Random Forest

Пример метрик точности:

Model	Accuracy
Random Forest	0.705
Logistic Regression	0.710
XGBoost	0.723
5. Results / Результаты

Выводы:

XGBoost показал наилучшие результаты (Accuracy = 0.723)

Random Forest и Logistic Regression дают близкие показатели (0.705 и 0.710)

Наименее точные прогнозы для редких позиций: LW, RW, RM, LM

Модели хорошо предсказывают популярные позиции: CB, CM, ST, GK

Дополнительно:

Важные признаки для предсказания позиции: overall, potential, stamina, short_passing, vision, interceptions и др.

6. Challenges / Сложности

Наличие пропусков в данных

Несбалансированность классов (редкие позиции плохо предсказываются)

Проблемы с масштабированием для Logistic Regression

7. Recommendations / Рекомендации

Применить балансировку классов (SMOTE, oversampling)

Масштабировать признаки для моделей, чувствительных к диапазону значений

Экспериментировать с гиперпараметрами XGBoost и Random Forest

В дальнейшем можно добавлять категориальные признаки, например, национальность или клуб

8. Visualizations / Визуализации

Распределение игроков по позициям

Матрица ошибок (Confusion Matrix)

ТОП-15 важных признаков для Random Forest

9. Conclusion / Заключение

Модель XGBoost показала наилучшие результаты для предсказания позиции игрока FIFA 26

Результаты подтверждают, что статистические показатели игроков (overall, potential, навыки) позволяют прогнозировать их основную позицию с высокой точностью (~72%)
